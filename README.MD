# Описание

Учебный проект, где я разобрался с интерфейсами в C#. 
Животное умеет двигаться, менять имя и издавать звуки.

## Интерфейсы:

- **IMovable** — движение.
- **IRenamable** — смена имени.
- **ISoundable** — голос.
